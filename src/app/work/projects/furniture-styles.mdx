## Overview

Development of a full-featured eCommerce platform for furniture products, designed to deliver a smooth and engaging online shopping experience. The platform enables users to explore furniture collections, view detailed product information, and securely place orders.

## Key Features

- **Product Catalog**: Dynamic listing of furniture products with categories, filters, and detailed product pages.
- **Shopping Cart System**: Add-to-cart, update quantities, and real-time price calculations.
- **Order Management**: Complete checkout workflow with order tracking and order history.
- **User Accounts**: User registration, login, and profile management system.
- **Admin Dashboard**: Backend panel for managing products, categories, inventory, and orders.
- **Responsive Design**: Mobile-friendly UI optimized for all screen sizes.

## Technologies Used

- **Django**: Backend framework for building secure business logic and APIs.
- **Bootstrap**: For responsive layout and prebuilt UI components.
- **JavaScript**: To handle dynamic front-end interactions.
- **MongoDB**: Used for flexible and scalable data storage.

## Challenges and Learnings

One of the main challenges was integrating Django with MongoDB while maintaining data consistency for orders and inventory. Performance optimization and handling real-time cart updates were key learning points. The project improved skills in database modeling and full-stack architecture.

## Outcome

The platform successfully launched in **2024** and provided a smooth, reliable online shopping experience for users. It helped streamline the online buying process for furniture businesses and improved customer engagement and conversion rates.
