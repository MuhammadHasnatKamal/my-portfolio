## Overview

Development of a smart menu recommender system designed to help restaurants manage their menus efficiently while providing personalized recommendations to customers. The system tracks customer orders, generates insights, and enables restaurants to optimize offerings and improve customer satisfaction.

## Key Features

- **Menu Management**: Administrators can create, update, and categorize menu items easily with a responsive interface.
- **Customer Order Tracking**: Records customer orders and generates insights for popular items and trends.
- **Personalized Recommendations**: Suggests menu items to customers based on their previous orders and preferences.
- **Analytics Dashboard**: Provides real-time reports and visualizations for restaurant managers to monitor performance.
- **Cloud-Hosted Backend**: Reliable server infrastructure to ensure high availability and smooth operations.

## Technologies Used

- **Next.js**: Front-end framework for building fast and interactive user interfaces.
- **TypeScript**: For strong type-checking and maintainable front-end code.
- **PostgreSQL**: Relational database for storing menus, orders, and customer data.
- **AWS**: Cloud infrastructure for hosting, storage, and scalability.
- **Supabase**: Backend-as-a-service platform for authentication, real-time updates, and API management.

## Challenges and Learnings

One key challenge was implementing personalized recommendations efficiently while maintaining fast query performance. Another challenge involved integrating Supabase and AWS services seamlessly to provide real-time updates for orders. The project strengthened skills in full-stack development, cloud deployment, and data-driven recommendation systems.

## Outcome

The system was successfully completed in **November 2025**, providing restaurants with a modern solution to manage menus, improve customer engagement, and make data-driven decisions. It enhanced operational efficiency, increased customer satisfaction, and delivered personalized dining experiences.
